Object rotation

rotation matrices are obtained by trig identity:

	/* about y-axis:

     cos(theta)     0    sin(theta)
     
        0           1         0
        
    -sin(theta)     0    cos(theta)

  */

  /* about x-axis:

        1         0           0

        0      cos(phi)    sin(phi)

        0     -sin(phi)    cos(phi)

  */

  /* about z-axis:

        cos(omega)  sin(omega)    0

       -sin(omega)  cos(omega)    0

            0           0         1

  */
